# Azure Data Engineering Projects

This repository contains sample Azure Data Engineering pipelines and frameworks developed using:

- Azure Data Factory (ADF)
- Azure Databricks
- PySpark
- Delta Lake
- Azure Synapse Analytics

## Sample Implementations
- Incremental data load framework using Delta Lake (SCD1 / SCD2)
- Metadata-driven ingestion framework
- API ingestion pipeline using Azure Data Factory
- Performance optimization techniques for Spark workloads
